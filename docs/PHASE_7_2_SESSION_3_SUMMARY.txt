## How to Use

### For Development
```dart
// Audit a question
final result = QuestionAuditService.auditQuestion(
  questionId: 'hom_1',
  skillId: 'skill_homophones',
  questionText: 'Which word means "a place to live"?',
  difficulty: 1,
  context: 'story',
);

// Check quality
print('Quality: ${result['qualityScore']}/100');
```
