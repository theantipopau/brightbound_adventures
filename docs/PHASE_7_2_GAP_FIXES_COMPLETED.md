## Module-by-Module Changes

### 1. Numeracy Module (+31 questions)

#### New Bank: DecimalQuestions (13 questions)
**Purpose**: Fill the critical decimal gap (was only 15 for Years 4-6)  
**ACARA Codes**: ACMNA079, ACMNA100, ACMNA127

**Levels**:
- **Level 1** (Difficulty 2): Decimal notation introduction
  - Understanding tenths place (2.5 → 5 tenths)
  - Basic decimal addition (0.3 + 0.4 = 0.7)
  - Decimal comparison (1.2 vs 1.7)
  - Simple subtraction (3.0 - 1.5 = 1.5)

- **Level 2** (Difficulty 3): Hundredths and ordering
  - Hundredths place understanding
  - Multi-decimal ordering (0.4, 0.04, 0.14)
  - Decimal addition with regrouping
  - Decimal multiplication (0.1 × 3)

- **Level 3** (Difficulty 4): Real-world applications
  - Money calculations ($5.00 - $2.45 = $2.55)
  - Multi-step addition (3.25 + 1.75 + 2.5)
  - Division application ($3.50 ÷ 5)
  - Estimation (7.8 + 2.1 ≈ 10)

**Sample Questions**:
- "A drink costs $2.45. You pay $5. How much change?" → Real-world context
- "Order from smallest: 0.4, 0.04, 0.14" → Critical comparison skill
- "6.4 - 2.8 = ?" → Advanced subtraction with regrouping

#### New Bank: DataInterpretationQuestions (10 questions)
**Purpose**: Address significantly under-represented data/statistics (was only 10)  
**ACARA Codes**: ACMSP069, ACMSP088, ACMSP111, ACMSP147

**Levels**:
- **Level 1** (Difficulty 2): Simple data reading
  - Reading lists of data
  - Comparing values
  - Finding ranges

- **Level 2** (Difficulty 3): Graph reading and analysis
  - Bar graph interpretation
  - Mode calculation
  - Median calculation

- **Level 3** (Difficulty 4): Statistical analysis
  - Mean/average calculation
  - Percentage interpretation (60% of 100)
  - Data spread understanding (Range)

**Sample Questions**:
- "From: Pizza 5, Burgers 3, Salad 2. How many like pizza?" → Basic reading
- "Class scores: 80, 85, 90, 85, 80. Mode?" → Mode concept
- "Average of (80 + 85 + 75 + 90) ÷ 4 = ?" → Mean calculation

#### New Bank: AdvancedGeometryQuestions (8 questions)
**Purpose**: Expand limited 3D geometry and transformation content  
**ACARA Codes**: ACMMG062, ACMMG087, ACMMG112, ACMMG143

**Levels**:
- **Level 1** (Difficulty 2): 3D shape identification
  - Sphere (soccer ball)
  - Cube (6 faces)
  - Cylinder (soup can)

- **Level 2** (Difficulty 3-4): 3D properties
  - Square pyramid vertices (5)
  - Reflection transformations
  - Volume calculations (5×3×2=30)
  - Edge counting (triangular prism → 9 edges)

**Sample Questions**:
- "How many faces does a cube have?" → 6
- "Volume: length 5cm, width 3cm, height 2cm?" → 30 cubic cm
- "How many vertices in square pyramid?" → 5

#### New Bank: HigherOrderThinkingNumeracyQuestions (4 questions)
**Purpose**: Address Bloom's Evaluate/Create gap  
**Cognitive Levels**: 5.0 (Evaluate), 6.0 (Create)

**Evaluate Questions** (Level 4-5):
- Compare solution methods (discount calculations)
- Verify mathematical reasoning (division checking)
- Assess reasonableness of averages

**Create Questions** (Level 6):
- Design a number story for 12 ÷ 3 = 4
- Create profit calculation (solve for quantity)
- Design a fair game with probability

**Sample Questions**:
- "Two discount methods: $5 off vs 20% off $25. Which saves more?" → Evaluation
- "Create a discount system: 3 items better value than 1 but worse than 4" → Creation
