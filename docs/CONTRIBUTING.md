# Contributing to BrightBound Adventures

Thank you for your interest in contributing to BrightBound Adventures!

## How to Contribute

1. **Report Bugs**: Open an issue describing the bug and how to reproduce it.
2. **Suggest Features**: Open an issue with a detailed description of the proposed feature.
3. **Pull Requests**:
    - Fork the repository.
    - Create a new branch for your feature or fix.
    - Ensure your code follows the existing style and passes all tests.
    - Submit a pull request with a clear description of your changes.

## Development Setup

See [DEV_SETUP.md](DEV_SETUP.md) for instructions on setting up your development environment.

## Content Guidelines

When adding new questions or educational content:
- Follow Australian English spelling conventions.
- Ensure content is age-appropriate (Ages 4-7).
- Align with NAPLAN / Australian Curriculum standards where applicable.
